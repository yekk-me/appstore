# TeslaMate

TeslaMate 是一个功能强大的特斯拉车辆数据自托管记录器，可以记录和可视化您的特斯拉车辆数据。

## 功能特性

- 📊 **数据记录**: 自动记录驾驶、充电、软件更新等数据
- 📈 **数据可视化**: 内置 Grafana 仪表板，提供丰富的数据图表
- 🔒 **隐私保护**: 数据完全存储在您的服务器上
- 🚗 **多车辆支持**: 支持同时记录多辆特斯拉车辆
- 📱 **Web 界面**: 提供友好的 Web 管理界面

## 安装要求

### 必要组件
- PostgreSQL 数据库服务（用于数据存储）

### 端口说明
- **TeslaMate**: 默认端口 4000，用于 Web 管理界面
- **Grafana**: 默认端口 3000，用于数据可视化仪表板

## 首次配置

### 1. 安装应用
在 1Panel 应用商店中找到 TeslaMate，点击安装并填写必要的配置信息。

### 2. 配置 TeslaMate
1. 安装完成后，访问 `http://你的服务器IP:端口` 进入 TeslaMate 管理界面
2. 按照界面提示进行特斯拉账号授权
3. 添加您的特斯拉车辆

### 3. 访问 Grafana 仪表板
1. 访问 `http://你的服务器IP:Grafana端口` 进入 Grafana
2. 使用安装时设置的管理员账号登录
3. 浏览预配置的仪表板查看车辆数据

## 数据安全

- 所有数据都存储在您的本地服务器上
- 支持数据加密存储
- 建议定期备份 PostgreSQL 数据库

## 故障排除

### 常见问题

1. **无法连接特斯拉账号**
   - 检查网络连接
   - 确认特斯拉账号和密码正确
   - 查看 TeslaMate 日志获取详细错误信息

2. **Grafana 无法显示数据**
   - 确认数据库连接正常
   - 检查 TeslaMate 是否成功收集到数据
   - 验证 Grafana 数据源配置

3. **服务启动失败**
   - 检查端口是否被占用
   - 确认数据库服务正常运行
   - 查看容器日志排查具体问题
